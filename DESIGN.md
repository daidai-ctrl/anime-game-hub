# DESIGN.md

## 气质与意象

深夜游戏桌面：显示器发出的冷光映在暗色墙面上，手柄搁在一旁，屏幕上滚动着游戏攻略。安静、专注、信息密度高。不是炫酷赛博朋克，而是玩家深夜刷攻略时那种"一屏找到答案"的实用感。

## 视觉策略

- 深色底色为主（`#0f1117` 近乎纯黑），减少眼疲劳，贴近游戏 Wiki 氛围
- 卡片与内容区域用微亮深灰（`#1a1d2e`）做层次区分
- 重点信息用青绿色（`#00d4aa`）高亮——暗示"解锁""兑换码"的游戏隐喻
- 图片和封面作为色彩来源，框架本身保持克制

## 配色方案

| 用途 | 色值 | 意象来源 |
|------|------|----------|
| 背景 | `#0f1117` | 深夜显示器边框 |
| 卡片/面板 | `#1a1d2e` | 暗处微微反光的桌面 |
| 边框 | `#2a2d3e` | 键盘缝隙的暗边 |
| 主强调色 | `#00d4aa` | 游戏中"激活""解锁"的绿色光效 |
| 次强调色 | `#6366f1` | 稀有装备的紫光 |
| 正文 | `#e2e8f0` | 屏幕上柔和的白字 |
| 次级文字 | `#94a3b8` | 游戏UI中的描述文字 |
| 成功/激活 | `#22c55e` | 兑换码有效 |
| 警告/过期 | `#ef4444` | 兑换码已过期 |

## 字体排版

- 西文：Inter（Google Fonts），清晰可读
- 标题：加粗 700，紧凑行高
- 正文：400/500，行高 1.75 保障长文阅读
- 代码/兑换码：等宽字体，带背景色块突出

## 动效与交互

- 极简：hover 状态用 `opacity` 和 `translateY(-2px)` 即可
- 折叠面板（FAQ）使用 Radix Accordion，无额外动画
- 禁止：粒子效果、滚动视差、3D 变换、视频背景

## 页面结构

- 两栏布局（桌面）：主内容 + 侧边栏（广告/TOC）
- 单栏布局（移动）：内容优先，广告插在段落间
- Hero 区域仅文字 + CTA，不用大图
- 卡片统一圆角 `8px`，间距 `16px`

## 设计禁忌

- 禁止浅色主题（本站纯深色）
- 禁止全屏视频/3D 背景
- 禁止复杂入场动画
- 禁止花哨字体
- 禁止无边框纯白卡片突兀出现在深色背景上
